# World Suicide Analysis

## 📊 Project Overview
This project analyzes global suicide data to identify trends, patterns, and key risk factors across countries, age groups, and gender. The analysis uses data visualization and analytics to better understand the factors influencing suicide rates and support data-driven public health strategies.

## 🎯 Objectives
- Analyze global suicide trends across different countries
- Identify high-risk age groups and gender differences
- Understand the relationship between economic factors and suicide rates
- Provide insights that can help improve mental health policies

## 📁 Dataset
Source: Kaggle  

Dataset contains information such as:
- Country
- Year
- Gender
- Age Group
- Population
- Number of Suicides
- Suicide Rate per 100k Population
- GDP per Capita
- Generation

Total Records: **27,000+ rows**  
Columns: **11+ features**

## 🛠 Tools & Technologies
- Power BI
- Data Cleaning
- Data Visualization
- Dashboard Creation

## ⚙️ Data Processing
- Imported dataset into Power BI
- Removed duplicate values
- Removed unnecessary columns
- Renamed columns for clarity
- Created a new feature: **GDP Category**

## 📈 Dashboard Insights
The dashboard provides visual insights on:
- Suicide trends across years
- Gender distribution
- Age group analysis
- Country comparison
- Economic factors affecting suicide rates

## 🔍 Key Findings
- Suicide rates are significantly higher among **men compared to women**.
- The **35–54 age group** shows the highest suicide rates globally.
- Higher GDP countries can still experience high suicide rates.
- Trends vary widely across different countries and generations.

## ✅ Conclusion
The analysis highlights suicide as a major global public health issue. Identifying high-risk groups and understanding socioeconomic influences can help governments and health organizations design better prevention strategies.
